1 HOW IT WORKS

Analysis Process

  1. Upload File: CSV or Excel file
  2. Select Columns: Map 4 required columns
  3. Customize: Colors, title, labels
  4. Generate Heatmaps: All facilities + Active only

Required Columns

  • hf_uid: HF001, HF002, FAC_123
  • year: 2023, 2024, 2025
  • month: 1-12 or 01-12
  • report: 1 (yes) or 0 (no)

Features

  • Two heatmaps: All facilities & active-only
  • All YYYY-MM x-axis labels shown
  • 300 DPI high-res PNG download
  • 15 color palettes (8 CB-friendly)
  • Custom title & axis labels
